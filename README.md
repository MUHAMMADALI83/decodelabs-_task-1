

 EDA & Feature Engineering Pipeline

This project performs **Exploratory Data Analysis (EDA) and **Feature Engineering** on a transactional dataset using Python. It includes data cleaning, missing value handling, outlier detection, feature engineering, one-hot encoding, correlation analysis, and runtime data validation to prepare data for machine learning.

 Features

* Data cleaning and preprocessing
* Missing value analysis
* IQR-based outlier detection
* Feature engineering
* One-hot encoding
* Correlation analysis
* Pandera schema validation

 Technologies

* Python
* Pandas
* NumPy
* SciPy
* Pandera
* OpenPyXL


```
pip install pandas numpy scipy pandera openpyxl
python eda_feature_pipeline.py
```

 Output

* Cleaned dataset
* Engineered features
* Validation report
* Correlation matrix
* Pipeline log
